# calculator v1.0
本工具直接使用python命令行实现

可能会使用到的特殊符号输入方法：

1. 乘方  
3 ^ 2 = 9

2. 开方  
sqrt(2) = 1.4142135623730951...

3. 三角函数   
sin(pi/6) = 0.5  
cos(pi/6) = 0.8660254037844387...  
tan(pi/6) = 0.5773502691896257...

4. 反三角函数  
asin(), acos, atan()同理

5. 对数函数
常用对数 log(10) = 1  
自然对数 ln(e) = 1

6. 特殊常量  
pi = 3.141592653589793...  
e = 2.718281828459045...

7. 位运算  
1 and 0 = 0
1 or 0 = 1
1 xor 0 = 1

还要注意一些东西：

1. 整数相除会向下取整  
3 / 2 = 1  
3.0 / 2 = 3 / 2.0 = 3.0 / 2.0 = 1.5  
3 / 2 * 0.9 = 0.9  
3 / 2.0 * 2 = 3

2. 请输入符合运算法则的式子
